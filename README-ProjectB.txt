Pages & views som implementerar nedan sidor  - - - - - - - - - 
Seed Page ---> 
    Hittas via sidan Seed där du också kan se en överblick av databasen
Overview Friends by country  ---> 
    Överblick av länder med vänner hittas på Home-sidan
Overview Friends and Pets in the cities of a specific country ---> 
    Genom att klicka vidare på See details under det land du vill veta mer om 
    så kommer du kunna se varje stads antal friends & pets i det landet 
List of all Friends in a Country and/or City ---> 
    I toppen av sidan kan du hitta List of friends för att få överblicken av alla vänner där du kan välja filtrering.
View the details of a Friend, including Address, Pets and Quotes ---> 
    Genom att klicka på den friend du är intresserad av i "List of friends" så kommer du vidare till detaljsidan.
Edit the details of a Friend with frontend and server-side validation --->
    Sidan för editering kan nås antingen via "list of friends" via knappen edit eller så kan sidan nås på edit iconen på detaljsidan
Edit the details of a Friends Address with frontend and server-side validation --->
    Sidan för editering kan nås antingen via "list of friends" via knappen edit eller så kan sidan nås på edit iconen på detaljsidan
Delete a Pet and a Quote from a Friend with modal confirmation --->
    Du kan göra detta via den röda Del knappen som finns brevid varje vän i "List of friends"
